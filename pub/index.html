What LESS and SASS do

LESS and SASS are preprocessors.

As preprocessors, they can extend the CSS language, adding very useful features to an otherwise very static language.

// LESS

@color-base-font : #343434;

div {
  background :
    lighten(@color-base-font, 50%)
    ;

  p {
    color : @color-base-font;
  }
}

// CSS

div {
  background : #b3b3b3;
}

div p {
  color : #343434;
}

Variables

@some-variable : some-value;

... and operations.

@nice-blue : #5B83AD;
#header { color : @nice-blue + #111; }

-> #header { color : #6c94be; }
